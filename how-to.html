How to use the Web Scraper tool:


  • IMPORTANT: Go to the CORS page and make sure the extension is downloaded and enabled.
  • Input the websites you wish you obtain data from.
  • Input the keywords you wish to use to search with.
  • Toggle the "Synonyms" button if you wish to search for synonyms in the case that the keyword is not found.
  • Click the "Scrape" button for instant results!
